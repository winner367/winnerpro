# 🧠 WinnerAI – Trading Dashboard

A simple trading dashboard UI built with **React**, **Vite**, and **Tailwind CSS**. Deployed using **GitHub Pages**.

## ✨ Features
- Sidebar with navigation
- Pages for:
  - 🤖 Trading Bots
  - 📈 Strategies
  - ⚙️ Settings
- Responsive & clean UI
